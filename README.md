# python-graphical-calculator
Simple Grapichal Calculator made with python's tkinter library.

### Operations
| Operation        | Symbol            |
| -------------    | -------------     |
| Addition         |  +                |
| Subtraction      |  -                |
| Multiplication   |  x                |
| Division         |  รท                |
| Floor Division   |  f                |
| Modulus          |  m                |
